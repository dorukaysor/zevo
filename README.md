# ZEVO — Power Everything.
```
A premium electronics e-commerce website built with HTML & CSS. Dark-themed & visually rich.
```


## Tech Stack

- **HTML5** — Semantic elements (`<nav>`, `<main>`, `<header>`, `<footer>`, `<section>`)
- **CSS3** — Custom properties, Flexbox, Grid
- **JavaScript** — Lenis smooth scroll (CDN) + nav scroll effect only
- **Fonts** — Google Fonts: Outfit (headings), Inter (body), JetBrains Mono (accents)
- **Icons** — Font Awesome 6 (CDN)

## How to Run

1. Install the **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**


## Design

- **Theme:** Dark mode (#0A0A0A background)
- **Accent:** Yellow (#FACC15)
- **Typography:** **Outfit** for headings, **Inter** for body, **JetBrains Mono** for labels
- **Currency:** INR / Rs.
- **Target viewport:** Desktop (1280px+)
